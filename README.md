# 使用指南

> [!note]
> 本文件面向使用者，帮助你快速上手这个统计学习 AI Agent。更底层的行为规则见 [[CLAUDE]]。

## 这是什么

- 这是一个用于统计学学习与知识沉淀的本地 Obsidian 工作区。
- Agent 会围绕 `statistics_mindmap.md`、学习进度、长期记忆和模块笔记来组织教学与记录。
- 学习内容默认按“描述统计 -> 概率基础 -> 推断统计 -> 高级应用”推进。

## 快速开始

1. 打开此仓库对应的 Obsidian Vault 或在 Trae 中继续对话。
2. 直接输入你的学习目标或下一步动作。
3. 完成一个小节后，用“总结”或“更新记忆”触发沉淀。

推荐起手句：

- `开始学习描述统计`
- `继续`
- `复习一下描述统计`
- `我想学习概率基础`
- `总结这一节`
- `更新记忆`

## 常用指令

| 你的说法 | Agent 默认行为 |
| --- | --- |
| `继续` | 按当前进度继续教学或继续当前维护任务 |
| `开始学习描述统计` | 进入对应模块，按知识地图推进 |
| `复习 xxx` | 回顾指定模块，结合当前进度补弱项 |
| `给我一道练习题` | 给出练习，通常先鼓励你作答，再反馈 |
| `总结` / `结束本节` | 生成本节总结，并更新学习记录 |
| `更新记忆` | 执行沉淀链路，更新 Session Log、Persistent Memory、Progress 和模块笔记 |
| `重置` | 默认执行软重置，只重置学习状态，不删除历史日志 |
| `清空学习记录` | 触发强重置候选，Agent 会先确认，再归档并重建 |
| `初始化` | 补齐最小学习结构，不主动删除历史 |

## 学习流程

1. Agent 先读取规则、知识地图、长期记忆、最近会话日志和学习进度。
2. 按当前模块进行讲解、举例、练习和必要的 Python 验证。
3. 你说“总结”“结束本节”或“更新记忆”后，Agent 执行沉淀。
4. 沉淀结果会写入学习日志、长期记忆、进度文件和模块笔记。

## 重置与初始化

### 软重置

- 适用：想从头重新学，但不想丢掉历史记录。
- 保留：`CLAUDE.md`、`statistics_mindmap.md`、`memory/User-Profile.md`、`memory/Session-Logs/`
- 重置：`Knowledge/Statistics/Progress.md`、`memory/Persistent-Memory.md` 中的学习状态、相关模块笔记状态

### 强重置

- 适用：想清空当前学习痕迹，重新初始化学习区。
- 默认流程：先归档到 `Inbox/Reset-Archive/YYYY-MM-DD/`，再重建核心学习文件。
- 注意：这是破坏性操作，执行前会再次确认。

### 初始化

- 适用：第一次搭建或补齐缺失文件。
- 行为：只补齐结构，不主动删除历史。

## 核心文件

- [[CLAUDE]]：Agent 系统规则与执行约定
- [[statistics_mindmap]]：统计学知识地图
- [[Progress]]：当前学习进度
- [[Persistent-Memory]]：长期记忆与稳定偏好
- [[Description-Statistics]]：描述统计主笔记
- `memory/Session-Logs/`：按日期记录会话摘要与学习轨迹

## 推荐使用方式

- 想学新内容：直接说模块名或目标，例如 `开始学习概率基础`
- 想延续上次内容：直接说 `继续`
- 想做练习：说 `给我一道关于偏度的练习题`
- 想整理本节：说 `总结` 或 `更新记忆`
- 想回到起点：说 `重置`
- 想彻底清空当前学习痕迹：说 `清空学习记录`

## 示例对话

### 进入新模块

```text
用户：开始学习描述统计
Agent：已加载记忆与地图：统计学知识地图已就绪，当前进度已同步。
```

### 学完后沉淀

```text
用户：总结这一节
Agent：生成结构化总结，并更新 Session Log、Progress、Persistent Memory 与模块笔记。
```

### 执行软重置

```text
用户：重置
Agent：你这次请求将按“软重置”处理：保留身份、偏好、规则与历史日志，只重置当前统计学习进度。
```

## 常见问题

### 为什么我说“继续”，有时感觉没接上？

- `继续` 会优先接续当前学习进度或最近一次明确任务。
- 如果上下文不够明确，Agent 会根据 `[[Progress]]`、最近 Session Log 和当前模块来推断。
- 想减少歧义，可以直接说：`继续描述统计里的关联部分`。

### 什么时候会自动沉淀？

- 当你说 `总结`、`结束本节`、`更新记忆` 时，会触发沉淀。
- 某些教学模块自然结束时，Agent 也会默认执行一次沉淀。
- 沉淀通常会更新 `memory/Session-Logs/`、`[[Persistent-Memory]]`、`[[Progress]]` 和对应模块笔记。

### 软重置后，哪些内容还在？

- 用户身份、偏好、系统规则、知识地图和历史日志默认保留。
- 被重置的主要是当前学习状态，例如 `[[Progress]]`、学习进度条目和模块状态。
- 如果你想连历史学习痕迹也一并清空，请明确说 `清空学习记录`。

### 初始化和重置有什么区别？

- `初始化` 是补齐最小可用结构，主要用于第一次使用或文件缺失。
- `重置` 是把当前学习状态恢复到起点，适合重新开始学习。
- `清空学习记录` 比 `重置` 更强，会进入强重置确认流程。

### 我可以直接指定学习顺序吗？

- 可以。
- 例如你可以直接说：`先跳过描述统计，直接学概率基础`。
- 如果你的要求与默认知识地图不一致，Agent 会优先按你的明确要求执行。

## 备注

- 默认中文输出。
- 默认一次优先解决一个问题。
- 如果你没有特别说明，Agent 会优先采取安全、可恢复的方式处理文件。
