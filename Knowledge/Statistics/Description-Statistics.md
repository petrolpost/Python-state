## 1. 数据类型

### 核心概念

| 类型     | 别名     | 特征                         | 例子               |
| -------- | -------- | ---------------------------- | ------------------ |
| 定类数据 | Nominal  | 只有类别，无顺序             | 血型、性别、城市   |
| 定序数据 | Ordinal  | 有类别，有顺序，间隔不等     | 会员等级、教育程度 |
| 定距数据 | Interval | 有顺序，间隔相等，无绝对零点 | 温度(°C)、年份     |
| 定比数据 | Ratio    | 有绝对零点，可进行比率运算   | 身高、体重、收入   |

### 层次关系

```
定类 → 定序 → 定距 → 定比
  ↑      ↑      ↑      ↑
最低   有顺序   等间隔   有绝对零点
```

### 数据类型 ↔ 图表选择

```
定类/定序数据 ──→ 频数表 / 饼图 / 条形图
定距/定比数据 ──→ 直方图 / 箱线图 / 散点图
```

## 2. 整理与展示

### 图表类型

| 图表   | 用途            | 适用数据类型     |
| ------ | --------------- | ---------------- |
| 频数表 | 计数汇总        | 定类/定序        |
| 条形图 | 类别对比        | 定类/定序        |
| 饼图   | 占比展示        | 定类（类别少时） |
| 直方图 | 连续数据分布    | 定距/定比        |
| 箱线图 | 5数概括、异常值 | 定距/定比        |
| 散点图 | 双变量关系      | 两个连续变量     |

### 箱线图 5数概括

```
最小值、Q1、中位数、Q3、最大值
```

### 偏度判断

| 类型         | 尾部分布 | 均值vs中位数  |
| ------------ | -------- | ------------- |
| 偏左（左偏） | 左尾更长 | 均值 < 中位数 |
| 偏右（右偏） | 右尾更长 | 均值 > 中位数 |
| 对称         | 两边均衡 | 均值 ≈ 中位数 |

## 3. 总结指标 - 集中趋势

### 三种代表值

| 指标   | 计算方法         | 特点           | 适用场景           |
| ------ | ---------------- | -------------- | ------------------ |
| 均值   | 所有值相加÷个数  | 受极端值影响   | 数据对称时         |
| 中位数 | 排序后中间值     | 不受极端值影响 | 偏态分布、有异常值 |
| 众数   | 出现次数最多的值 | 可能不唯一     | 定类数据           |

### 极端值影响

```
正常数据: 21, 22, 23, 24, 25 → 均值=23
有异常值: 21, 22, 23, 24, 100 → 均值=38（被拉高），中位数=23（稳定）
```

---

## 4. 总结指标 - 离散程度

### 四大指标对比

| 指标   | 计算公式        | 特点                        | 易受极端值影响？ |
| ------ | --------------- | --------------------------- | ---------------- |
| 极差   | 最大值 - 最小值 | 最简单，只看两端            | ✅ 是             |
| IQR    | Q3 - Q1         | 中间50%范围，不受极端值影响 | ❌ 否             |
| 方差   | Σ(值-均值)² ÷ n | 平方消除正负抵消            | ✅ 是             |
| 标准差 | √方差           | 方差的平方根，还原原单位    | ✅ 是             |

### IQR 异常值检测

```
异常值定义：> Q3 + 1.5×IQR 或 < Q1 - 1.5×IQR
```

### 四分位数计算（位置公式法）

```
数据：2, 4, 6, 8, 10, 12, 14（n=7）
Q1位置 = (n+1) × 0.25 = 8 × 0.25 = 2 → Q1=4
Q3位置 = (n+1) × 0.75 = 8 × 0.75 = 6 → Q3=12
IQR = 12 - 4 = 8
```

## 5. 总结指标 - 形状

### 偏度（Skewness）

| 类型 | 别名      | 尾部分布   | 均值vs中位数  | 例子                       |
| ---- | --------- | ---------- | ------------- | -------------------------- |
| 偏左 | 左偏/负偏 | 左尾更长   | 均值 < 中位数 | 大部分人及格，少数不及格   |
| 偏右 | 右偏/正偏 | 右尾更长   | 均值 > 中位数 | 大部分人低收入，少数高收入 |
| 对称 | 无偏      | 两边差不多 | 均值 ≈ 中位数 | 接近正态分布               |

### 峰度（Kurtosis）

| 类型 | 特征                       | 例子         |
| ---- | -------------------------- | ------------ |
| 尖峰 | 峰又高又窄，数据集中在中心 | 高考成绩分布 |
| 平峰 | 峰又低又平，数据分散       | 均匀分布     |

**判断**：峰度>0为尖峰，<0为平峰，=0为正态峰度

## 6. 总结指标 - 关联

### 协方差

```
协方差 = Σ(X-X̄)(Y-Ȳ) ÷ n

协方差 > 0：正相关
协方差 < 0：负相关
协方差 ≈ 0：不相关
```

### 相关系数

```
r = 协方差 / (σX × σY)

-1 ≤ r ≤ 1
|r| < 0.3：弱相关
0.3 ≤ |r| < 0.7：中等相关
|r| ≥ 0.7：强相关
```
