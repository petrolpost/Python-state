## 1. 数据类型

### 核心概念

| 类型 | 别名 | 特征 | 例子 |
|-----|------|------|------|
| 定类数据 | Nominal | 只有类别，无顺序 | 血型、性别、城市 |
| 定序数据 | Ordinal | 有类别，有顺序，间隔不等 | 会员等级、教育程度 |
| 定距数据 | Interval | 有顺序，间隔相等，无绝对零点 | 温度(°C)、年份 |
| 定比数据 | Ratio | 有绝对零点，可进行比率运算 | 身高、体重、收入 |

### 层次关系

```
定类 → 定序 → 定距 → 定比
  ↑      ↑      ↑      ↑
最低   有顺序   等间隔   有绝对零点
```

### 数据类型 ↔ 图表选择

```
定类/定序数据 ──→ 频数表 / 饼图 / 条形图
定距/定比数据 ──→ 直方图 / 箱线图 / 散点图
```

### 练习与答案

| 题目 | 答案 |
|-----|------|
| 血液类型 | 定类 |
| 教育程度 | 定序 |
| 考试成绩 | 定距/定比（取决于业务定义）|
| 月薪 | 定比 |

## 2. 整理与展示

### 图表类型

| 图表 | 用途 | 适用数据类型 |
|-----|------|------------|
| 频数表 | 计数汇总 | 定类/定序 |
| 条形图 | 类别对比 | 定类/定序 |
| 饼图 | 占比展示 | 定类（类别少时）|
| 直方图 | 连续数据分布 | 定距/定比 |
| 箱线图 | 5数概括、异常值 | 定距/定比 |
| 散点图 | 双变量关系 | 两个连续变量 |

### 箱线图 5数概括

```
最小值、Q1、中位数、Q3、最大值
```

### 偏度判断

| 类型 | 尾部分布 | 均值vs中位数 |
|-----|--------|-------------|
| 偏左（左偏） | 左尾更长 | 均值 < 中位数 |
| 偏右（右偏） | 右尾更长 | 均值 > 中位数 |
| 对称 | 两边均衡 | 均值 ≈ 中位数 |

### Python 验证

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# 频数表
scores = [85, 92, 78, 88, 95, 73]
freq_table = pd.Series(scores).value_counts().sort_index()
freq_table.plot(kind='bar')

# 箱线图
q1, median, q3 = np.percentile(scores, [25, 50, 75])
# 偏度
from scipy import stats
print(stats.skew(scores))
```

## 3. 总结指标 - 集中趋势

### 三种代表值

| 指标 | 计算方法 | 特点 | 适用场景 |
|-----|--------|------|---------|
| 均值 | 所有值相加÷个数 | 受极端值影响 | 数据对称时 |
| 中位数 | 排序后中间值 | 不受极端值影响 | 偏态分布、有异常值 |
| 众数 | 出现次数最多的值 | 可能不唯一 | 定类数据 |

### 极端值影响示例

```
正常数据:  21, 22, 23, 24, 25
            ├─ 均值 = 23

有异常值:  21, 22, 23, 24, 100
            ├─ 均值 = 38   ← 被拉高！
            └─ 中位数 = 23  ← 依然稳定
```

### 练习与答案

**练习1** - 数据集A：2, 3, 3, 5, 7, 8, 9
- 均值 = 5.28
- 中位数 = 5
- 众数 = 3

**练习2** - 数据集B：21, 22, 23, 24, 25, 100
- 选择中位数（因均值被极端值影响）
