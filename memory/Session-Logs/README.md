# Session Logs 使用说明

- 文件命名：`YYYY-MM-DD-session.md`
- 写入时机：用户说“总结 / 沉淀 / 结束本节”或会话结束
- 最小内容：关键讨论点、练习结果、下一步、关联链接

建议使用以下 Frontmatter：

```yaml
---
date: YYYY-MM-DD
tags: [session-log, statistics]
related: [[Persistent-Memory]], [[Progress]]
---
```
